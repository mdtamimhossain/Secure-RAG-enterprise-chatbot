# Codemars Corporation IT Security Basics

Department: General
Audience: All employees
Confidentiality: Internal

This document explains security practices that every Codemars employee must follow.

## Account Security

All employees must use multi-factor authentication for MarsID, OrbitDocs, GitHub Enterprise, and finance systems. MFA reset requests must be submitted through MarsDesk and approved by the employee's manager.

## Device Rules

Company laptops must use full-disk encryption, automatic screen locking after 10 minutes, and endpoint protection managed by IT. Employees may not store company documents on personal cloud drives.

## Data Handling

General company documents may be discussed internally. HR, finance, executive, legal, and customer-confidential materials must only be discussed with authorized roles.

## Reporting

Suspicious emails, lost devices, or suspected account compromise must be reported to security@codemars.example within one hour.
